## Telegram Bot Api 1.0.0 ([6.1](https://core.telegram.org/bots/api#june-20-2022))

##### Требования:

+ `PHP >= 5.6`
+ `ext-curl`
+ `ext-mbstring`
+ `ext-json`


##### Установка:

```
composer require mnlnk/tgbot-api
```
